- **错落有致的布局** - 不对称但平衡
- **舒适的间距** - 呼吸感的设计

### 🛠️ Alexander Obenauer (alexanderobenauer.com)
- **工具感** - 整洁、有序、功能导向
- **清晰的层次** - 信息组织井然

## ✨ 设计亮点

### 1. **书籍式超大留白**
```
最大宽度: 896px (4xl)
上方留白: 128px (pt-32)
段落间距: 96px (mb-24)
```

每个元素都有充足的呼吸空间，像在阅读精装书。

---

### 2. **不对称网格布局**
```
左侧（7列）: 主标题 + 描述
右侧（5列）: 导航卡片
```

打破传统对称，但视觉平衡。参考 maggieappleton 的布局。

---

### 3. **字体层级清晰**

**标题：**
- 思源宋体，5xl - 7xl (48px - 72px)
- 行高 1.1（紧凑有力）

**正文：**
- 思源黑体，lg (18px)
- 行高 1.9（极度舒适的阅读体验）
- 最大宽度 32em（最佳阅读宽度）

**等宽字体：**
- JetBrains Mono
- 用于日期、标签、元数据

---

### 4. **微妙的交互**

**导航卡片 hover:**
- 边框变红
- 箭头淡入并移动
- 标题变色
- 轻微阴影

**文章列表 hover:**
- 标题变红
- 保持克制，不过度

**动画时序:**
- 渐进式出现（stagger）
- 每个元素间隔 0.1-0.2s
- 缓动曲线舒适

---

### 5. **像书的元素**

**章节标记：**
```
Digital Garden · 数字花园
```
小型等宽字体，大写，字母间距加大

**出版信息：**
```
最近更新：2026.05.12 · 持续培育中
```
等宽字体，小字号，像书的版权页

**目录式列表：**
```
05.12  关于 Agent 系统设计的几点思考  [Essay]
```
日期 + 标题 + 类型，清晰的视觉层次

---

### 6. **数字花园的隐喻**

**用词选择：**
- "生长" 而非 "发布"
- "培育中" 而非 "开发中"
- "Recent Updates" 而非 "Latest Posts"
- "Tended with care" 而非 "Copyright"

**导航数字：**
```
Blog    24   ← 像标记植物数量
Notes   156
Tools   12
```

## 🎨 配色方案

保持原有的杂志编辑部配色：

```css
背景: #FAF8F5 (暖白)
文字: #1A1A1A (近黑)
辅助: #6B6560 (暖灰)
强调: #C4463A (朱砂红，<5%使用)
边框: #E8E3DD (浅暖灰)
```

**特点：**
- 纸张质感
- 温暖舒适
- 不刺眼
- 长时间阅读友好

## ✅ 特点总结

**这个设计的核心是：**

1. **慢下来** - 不追求视觉冲击，而是舒适阅读
2. **留白** - 让内容呼吸
3. **克制** - 只在需要的地方强调
4. **有机** - 内容像植物一样生长
5. **工具** - 实用至上

**一句话描述：**
> 像翻阅一本精装书，在书页间漫步于数字花园。
